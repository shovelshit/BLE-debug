# BLE 调试助手

一款微信小程序版的低功耗蓝牙（BLE）调试工具，用于开发和测试蓝牙设备，可替代 nRF Connect、LightBlue 等 PC 工具。

## 功能

### ⚡ 快捷操作（主页）
- 预配置「设备 MAC + Service UUID + 特征值 UUID + 写入值」组合，保存为卡片
- 一键发送：自动扫描 → 连接设备 → 发现 GATT 服务 → 写入数据，全程无需手动操作
- 支持 HEX / 文本两种写入格式
- 多卡片并发安全（会话令牌机制，切换设备时自动断开旧连接）
- 卡片支持新增、编辑、删除，颜色自定义

### 📡 扫描页
- 扫描周边 BLE 设备，实时更新信号强度（RSSI 条状图）
- 搜索栏：按设备名称或 MAC 地址过滤
- 可过滤无名称设备，按信号强度排序
- 点击设备自动连接并进入详情页

### 🔌 设备详情页
- 展示所有 GATT 服务和特征值
- 特征值属性以 badge 标注：`Read` / `Write` / `Notify` / `WriteNoResp`
- 支持一键复制 Service UUID、Characteristic UUID
- 快速跳转到快捷操作页预填当前特征值

### ✏️ 特征值操作页
- **Write**：写入 HEX 或文本数据，支持预设值管理（增删改）
- **Read**：读取当前特征值
- **Notify**：订阅 / 取消订阅通知，实时接收推送数据
- 操作日志面板，记录收发数据和错误信息
- 写入失败自动重试（含配对流程，降级为 writeNoResponse）

### 📋 日志页
- 全局蓝牙操作日志，记录所有收发数据与错误

## 使用要求

- **平台**：iOS 或 Android 手机微信（Mac / PC 微信不支持蓝牙 API）
- **调试方式**：微信开发者工具 → 「真机调试」扫码在手机上运行

### 技术栈
- 微信小程序原生框架（WXML / WXSS / JS）
- 微信蓝牙 API（`wx.openBluetoothAdapter` 等）
- 本地持久化：`wx.setStorageSync`

### 目录结构

```
├── app.js              全局状态、蓝牙适配器初始化、日志管理
├── app.json            页面注册、TabBar 配置
├── pages/
│   ├── quick/          快捷操作页（主页）
│   ├── index/          设备扫描页
│   ├── device/         设备详情页
│   ├── characteristic/ 特征值操作页
│   └── log/            日志页
└── assets/icons/       TabBar 图标
```

## 个人微信小程序搭建思路

### 1. 注册账号

1. 访问 [mp.weixin.qq.com](https://mp.weixin.qq.com)，点击「立即注册」→ 选择「小程序」。
2. 使用**个人邮箱**完成注册（无需营业执照），服务类目选择「工具 > 效率工具」或「开发者工具」。
3. 注册完成后在「开发 → 开发管理 → 开发设置」中获取 **AppID**（形如 `wx1234abcd`）。

> ⚠️ 个人主体小程序有部分限制：不支持微信支付、不支持部分敏感接口，但蓝牙 API 完全可用。

### 2. 安装开发工具

1. 下载并安装 [微信开发者工具](https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html)（建议 Stable 版）。
2. 打开工具 → 扫码登录 → 新建项目 → 填入 AppID 并选择本项目目录。

### 3. 配置项目

将 `project.config.json` 中的 `appid` 替换为你自己的 AppID：

```json
{
  "appid": "wx你的AppID",
  "projectname": "BLE调试助手"
}
```

### 4. 本地调试

- **模拟器调试**：蓝牙 API 在 PC 模拟器中无法使用，只能看 UI 布局。
- **真机调试**（推荐）：点击工具栏「真机调试」→ 手机微信扫码 → 在手机上实时运行并查看日志。
- **预览**：点击「预览」→ 扫码 → 手机上运行（无实时日志）。

### 5. 申请蓝牙权限

小程序蓝牙 API 无需额外后台审批，但真机运行时系统会弹出「允许访问蓝牙」授权弹窗，用户同意后即可使用。

### 6. 发布上线（可选）

1. 开发者工具 → 上传代码 → 填写版本号和备注。
2. 登录 [mp.weixin.qq.com](https://mp.weixin.qq.com) → 版本管理 → 提交审核。
3. 个人小程序审核通过后默认**不可搜索**，需通过二维码或好友分享访问。
4. 若需要公开搜索，需完成「微信认证」（个人主体暂不支持认证，可考虑升级为个体工商户主体）。

### 7. 蓝牙调试注意事项

| 环境 | 蓝牙 API 可用 | 说明 |
|------|-------------|------|
| 微信开发者工具（PC 模拟器） | ❌ | 无蓝牙硬件访问权限 |
| Mac / PC 微信客户端 | ❌ | 微信桌面版不开放蓝牙 API |
| iOS 微信 App（真机调试） | ✅ | 需授权蓝牙权限 |
| Android 微信 App（真机调试） | ✅ | 需授权蓝牙 + 位置权限 |
